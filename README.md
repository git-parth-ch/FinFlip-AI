# 💸 FinFlip AI 
> **AI-Powered Financial Independence for Gen Z.**

FinFlip AI is a frictionless, conversational personal finance app designed for students and young professionals (18-23) in India. By combining a natural language chat interface with bite-sized financial education, FinFlip cures "budgeting paralysis" and builds long-term wealth habits without requiring complex spreadsheets.

---

## 🚧 Current Project Status: Frontend MVP
**Note to Judges / Reviewers:** This repository currently contains the **Basic Frontend Prototype**. The UI/UX architecture, component designs, and navigation flow have been implemented. 

The backend (REST APIs, ML Categorization Engine, and LLM integration) is currently under active development and will be integrated in the next phase.

---

## ✨ Core Features (UI Implemented)

* **💬 Conversational Expense Tracking:** A frictionless chat interface that replaces intimidating ledgers. Users simply type "Spent ₹150 on auto."

* **📊 Contextual AI Nudges:** A visually clean insights dashboard providing judgment-free financial advice (e.g., "Cooking tonight keeps you on track to save ₹2,000 this month!").
* **🔒 Secure Sync Onboarding:** Premium UI flows preparing for our Phase 2 integration with India's RBI-regulated Account Aggregator (AA) framework.

---

## 💻 Tech Stack

### 📱 Frontend (Current)
* **Framework:** React Native (Cross-platform iOS & Android)
* **Language:** TypeScript (`.tsx`) for strict typing and scalable architecture.
* **Styling:** Custom minimal CSS / Tailwind (Dark mode optimized)
* **Prototyping:** Figma & Google Stitch

### ⚙️ Backend & AI Architecture (Planned / WIP)
* **API Layer:** REST APIs (Java / Python FastAPI)
* **Database:** PostgreSQL (Relational DB for ACID-compliant financial accuracy)
* **Categorization Engine:** High-speed Python ML Classifier (XGBoost / Random Forest) for instant merchant tagging.
* **Conversational AI:** GPT-based LLM with strict sandboxing for personalized financial guidance.
* **Infrastructure:** AWS
